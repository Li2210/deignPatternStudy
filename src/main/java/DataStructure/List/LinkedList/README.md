# Java中的集合
LinkedList in java
------
List都是有序的  
LinkedList是用链表来存储数据,适合数据的动态插入和删除,随机访问和遍历速度比较慢。  
提供了List接口中没有定义的方法,专门用于操作表头和表尾元素,可以当作堆栈,队列和双向队列使用。
